Team Management GraphQL API

Welcome to the Team Management GraphQL API! This API allows you to manage teams, squads, players, coaches, leagues, tournaments, venues, and matches for your sports organization.

Features:
  • CRUD operations for squads, players, coaches, leagues, tournaments, venues, and matches
  • GraphQL endpoint for flexible queries and mutations
  • Pagination, filtering, and ordering support
  • Rich type system with Strawberry GraphQL
Getting Started:
  1. Access the GraphQL Playground to explore and test queries.
  2. Read the API Documentation for schema details and usage examples.
Example Query:
{
    squads {
        id
        name
        size
        league {
            name
        }
    }
}

For more information, see the documentation or contact the API maintainer.
